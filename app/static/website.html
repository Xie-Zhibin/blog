功能介绍

不得不说，前端的飞速发展让让我们受益颇丰，各种前端UI框架、JavaScript库不断涌现，在很大程度上方便了像我一样并不太了解前端技术的后端开发者。

经过一番选择，UI框架选用了国产的Amaze UI,主要原因是它足够轻量级，有足够的组件给我搭建。也曾想过用Bootstrap,但后来发现它对我来说体积过于庞大、功能过于强大，我等并不能很好得驾驭，遂决心放弃Bootstrap选用国产的Amaze UI。

Amaze UI 是一个开源的HTML5跨屏前端UI框架，对于浏览器兼容性，经作者测试，能很好地兼容Chrome Firefox Safari三大主流浏览器，基本支持一些国产浏览器，因此也在此建议用户使用主流浏览器访问本网站以获得更好的体验。另外，虽然Amaze UI写了响应式，支持移动端，但由于有部分布局和js是作者自己写的，没有过多地考虑浏览器兼容问题，所以有些页面在移动端可能会出现问题，所以也不建议用移动端访问本站。使用UI框架好处在于可以以搭积木的形式拼接好每一个网页，修改部分只在少数，并且不用去考虑让每一个前端开发者都头疼的浏览器兼容性问题，大大节省了我的时间，当然，很显著的弊端就是不能很好地去设计自己真正想要的页面。

Handlebars是一个很好的JavaScript语义模板库，通过对view和data的分离来快速构建Web模板，非常容易上手，对JS基本不懂的我都能轻易使用。用这个库最大的好处在于可以让一个人完全实现前后端分离，后台功能全部由api的形式给出，如此大大减少了后端的任务，但同时也加大了前端工作难度，最重要的是在一定程度上提高了用户体验。

如此一来，很多功能都被搬到了客户端，以致服务端的压力可能会小于客户端。估计了一下，整个网站的JS代码可能已经超过了Python代码，主要是用了较多的ajax异步处理，然而作者写的JS实在太烂，相信可以优化的空间还有很多，欢迎诸位指出。

后端语言，框架，数据库没有别的选择，因为一直学的都是 Python + Flask + MySQL 做web后端，暂未尝试过其他工具。

网站服务器采用了腾讯云服务器。为了提高加载速度，图片等静态资源存放在七牛云，juqery handlebars 等js类库在 Bootstrap中文网开源项目免费 CDN 服务 中下载。

网站logo由作者的一个学习绘画设计的朋友专门设计，在此特别鸣谢。

最后希望诸位能在这里发现一些有价值的东西，谢谢。
网站初衷

从去年就有搭建个人网站的想法，也尝试着写过原型，苦于当时能力有限，很多功能不能实现。恰逢五一三天假期，遂决定趁此机会实施这个"蓄谋已久"的计划。

搭建这个网站的主要目的是写文章与诸位分享交流，同时实践和检验现在所学的知识。
